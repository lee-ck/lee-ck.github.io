What is Model Predictive Control (MPC) ?

{: .align-center}

[Image from Prof.Rolf Findeisen's slides]

Pros

  • Proactive behavior
  • State and input constraints
  • Any model
  • Any objective
  • ...

Cons

  • Computational complexity
  • May not be feasible
  • May not be stable
  • Unceratinty
  • ...

Stability of Linear MPC [Ref. B2]

Assumptions

다음의 4가지 가정을 통하여 Stability를 보장할 수 있음.

Assumption 1.
Stage cost qqq is positive definite. i.e. it is strictly positive and only zero at the origin.

q(xk,uk)=xk⊤Qxk+uk⊤Rukp(xN)=xN⊤PxN\begin{aligned} q(x_k,u_k)&= x_k^\top Q x_k + u_k^\top R u_k \\ p(x_N) &= x_N^\top P x_N \end{aligned}q(xk​,uk​)p(xN​)​=xk⊤​Qxk​+uk⊤​Ruk​=xN⊤​PxN​​

Assumption 2.
Terminal set is invariant under the local control law v(xk)=Kxkv(x_k)=Kx_kv(xk​)=Kxk​ satisfying:

xk+1=Axk+Bv(xk)∈Xf, for all xk∈Xfx_{k+1} = Ax_k + Bv(x_k) \in \mathbb{X}_f , \ \text{for all} \ x_k \in \mathbb{X}_fxk+1​=Axk​+Bv(xk​)∈Xf​, for all xk​∈Xf​

Assumption 3.
Terminal cost is a continuous Lyapunov function in the terminal set Xf\mathbb{X}_fXf​ satisfying:

p(xk+1)−p(xk)≤−q(xk,uk), for all xk∈Xfp(x_{k+1}) - p(x_k) \leq -q(x_k,u_k), \ \text{for all} \ x_k \in \mathbb{X}_fp(xk+1​)−p(xk​)≤−q(xk​,uk​), for all xk​∈Xf​

Assumption 4.
Assume that the MPC problem has a feasible solution at the initial time.

How to Compute Terminal Set and Cost

Unconstrained LQR control law를 활용:

K=−(B⊤PB+R)−1B⊤PAP=A⊤PA+Q−A⊤PB(B⊤PB+R)−1B⊤PA\begin{aligned} K &= -(B^\top PB + R)^{-1}B^\top P A \\ P &= A^\top P A + Q - A^\top P B(B^\top PB + R)^{-1} B^\top PA \end{aligned}KP​=−(B⊤PB+R)−1B⊤PA=A⊤PA+Q−A⊤PB(B⊤PB+R)−1B⊤PA​

P는 discrete-time algebraic Riccati equation의 해.

위의 KKK와 PPP로 Assumption 3 를 만족시킬 수 있음.

p(xk+1)−p(xk)=xk+1⊤Pxk+1−xk⊤Pxk=xk⊤(−P+A⊤PA−A⊤PB(B⊤PB+R)−1B⊤PA−K⊤RK)xk=−xk⊤Qxk−uk⊤Ruk\begin{aligned} p(x_{k+1}) - p(x_k) &= x_{k+1}^\top P x_{k+1} - x_{k}^\top P x_{k} \\ &= x_k^\top (-P + A^\top PA - A^\top PB(B^\top PB + R)^{-1}B^\top PA - K^\top RK )x_k \\ &= -x_k^\top Q x_k - u_k^\top R u_k \end{aligned}p(xk+1​)−p(xk​)​=xk+1⊤​Pxk+1​−xk⊤​Pxk​=xk⊤​(−P+A⊤PA−A⊤PB(B⊤PB+R)−1B⊤PA−K⊤RK)xk​=−xk⊤​Qxk​−uk⊤​Ruk​​

여기서 계산한 K를 활용하여 Terminal set은 다음과 같이 계산할 수 있음.

xk+1=(A+BK)xk∈Xf, for all xk∈Xfx_{k+1} = (A+BK)x_k \in \mathbb{X}_f, \ \text{for all} \ x_k \in \mathbb{X}_fxk+1​=(A+BK)xk​∈Xf​, for all xk​∈Xf​

Invariant set은 Minkowski sum을 통하여 연산 가능 (MPT3, CORA 등의 software 사용)

Polytope, Zonotope, Box, Circle, etc 형태로 계산이 가능.

이때 다음의 조건은 반드시 만족해야함.
Xf⊆X,Kxk∈U, for all xk∈Xf\mathbb{X}_f \subseteq \mathbb{X}, Kx_k \in \mathbb{U}, \ \text{for all} \ x_k \in \mathbb{X}_fXf​⊆X,Kxk​∈U, for all xk​∈Xf​
